Place Mode


The GUI can be set to Place mode by pressing 'P' key. Places can be added to the active tab by right clicking the mouse. By left clicking the mouse a menu of two options appear. The first option is add label and the second is edit tokens. With the first option it is possible to edit or add the label. The second option enables the user to change the number of tokens in the right clicked place.
By left clicking anywhere in side the drawing area a new place will be added at that place.
